Sitio Web Casa de Oración

El proyecto Casa de Oración es una página web diseñada profesionalmente para ofrecer información detallada sobre un templo, el objetivo principal es proporcionar a los usuarios un espacio accesible y sencillo donde puedan conocer más sobre los servicios, horarios, eventos y actividades del templo.

La plataforma fue desarrollada utilizando HTML, CSS y JavaScript, garantizando una experiencia de usuario fluida y visualmente atractiva, el sitio cuenta con una estructura responsiva, lo que permite que los usuarios accedan cómodamente desde cualquier dispositivo, ya sea un ordenador de escritorio, tablet o móvil.

Este proyecto no solo demuestra mis habilidades técnicas en desarrollo web, sino también mi capacidad para trabajar en proyectos con objetivos específicos y crear soluciones que sean tanto funcionales como atractivas visualmente.
